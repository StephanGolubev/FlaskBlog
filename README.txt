Простой сайт для размещения постов
Сделан на микрофреймворке Flask
База данных - SQLite

Чтобы запустить:
1. pip install virtualenv
2. python -m virtualenv env
3. env\Scripts\activate
4. python -m pip install -r requirements.txt
5. cd app
6. python init_db.py
7. flask run

Ссылка на веб приложение:
http://127.0.0.1:5000/

Ссылка на веб приложение из index.html:
http://127.0.0.1:5000/hello
